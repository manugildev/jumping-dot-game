# Jumping Dot Game
[![](http://i.imgur.com/07oXoTC.png "jumping dot")](http://bit.ly/jumpingdot)
DOWNLOAD THE GAME: http://bitly.com/jumpingdot (Google Play)

This game is an Obstacle avoidance game made in Java using Eclipse IDE. The aim of the player is to avoid box controlling the ball just with the finger and get the best highscore.
